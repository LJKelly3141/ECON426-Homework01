# ECON 426: Homework 01 - Data Handling

## Assignment Overview
This assignment practices data handling with R, including loading, viewing, and summarizing data.

## Files
- `Homework01.qmd` - Quarto document with assignment instructions
- `data/cholesterol.csv` - Dataset for the assignment
- `_quarto.yml` - Quarto project configuration

## Instructions
1. Open this project in RStudio
2. Open `Homework01.qmd`
3. Complete the assignment by following the instructions
4. Render to MS Word by clicking "Render" or running `quarto render Homework01.qmd`
5. Submit the generated `.docx` file to Canvas

## Rendering
To render the document to MS Word:
```r
quarto::quarto_render("Homework01.qmd")
```

Or use the Render button in RStudio.
